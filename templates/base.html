Instructions

This app will only work with CSV files that are in the same format as the test file. The 'abundance' columns can be excluded, but the results obtained will be limited, as explained bellow.

Taxa names must be an exact match of the ones from the reference spreadsheet. If some of the names do not match, they will not be included in the calculation.

PSI, E-PSI and LIFE related indices require abundance values. WHPT related indices although do not require, their result will depend on the abundance values.

Only adults of the Gyrinidae family should be used for calculation of the DEHLI index. If the sample contains specimens of Gyrinidae family, the CSV file should be adjusted accordingly and uploaded as 'DEHLI' file. Otherwise, upload the regular CSV file for both 'full-CSV' and 'DEHLI'.

Refresh the page to restart the app.
